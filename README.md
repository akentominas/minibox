# 🎯 minibox 
A minimal container runtime written in Go to learn Linux internals like namespaces, cgroups, and process isolation.

## 🧱 Prerequisites

### 🐧 Linux Environment (REQUIRED)

You must run this on Linux.

Recommended setup:

VM using UTM (Mac users)
Distribution:
Fedora ✅
AlmaLinux ✅
Ubuntu ✅

👉 macOS alone will NOT work (namespaces are Linux-only)
